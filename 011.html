활용법 예시

다양한 Proxmox 활용 사례로 홈서버의 가능성을 확장하세요

개인 NAS 구축

TrueNAS나 OpenMediaVault를 VM으로 설치하거나 ZFS 기능 활용하여 안정적인 파일 서버 구축

미디어 서버

Plex나 Jellyfin을 설치하여 집 안 어디서든 개인 미디어 라이브러리에 접근

스마트 홈 허브

Home Assistant로 스마트 기기들을 통합 제어하는 허브 구축

광고 차단

Pi-hole과 같은 DNS 기반 광고 차단 소프트웨어로 네트워크 전체 광고 차단

개발 샌드박스

OS 테스트와 개발 환경을 안전하게 실험할 수 있는 격리된 환경 제공

네트워크 랩

가상 네트워크 토폴로지 구성, 방화벽 테스트, 라우팅 프로토콜 학습

GPU 서버

AI 모델 실행, Ollama 설치로 Llama 3.1 등의 언어 모델을 로컬에서 구동

각 사용 사례는 VM(Virtual Machine)이나 LXC(Linux Container)로 구현하여 하나의 서버에서 동시에 운영 가능합니다!
