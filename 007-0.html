Proxmox란?

Proxmox Virtual Environment (VE)

리눅스 기반의 오픈소스 가상화 플랫폼으로, 두 가지 핵심 기술(KVM과 LXC)을 결합하여 강력하고 유연한 가상화 환경을 제공합니다.

Type 1 하이퍼바이저 오픈소스 웹 관리 인터페이스 고성능

핵심 가상화 기술

KVM (Kernel-based Virtual Machine)

  • 완전한 가상 컴퓨터 생성 기술
  • Windows, Linux 등 모든 OS 실행 가능
  • 높은 보안성과 완전한 격리
  • 더 많은 시스템 자원 사용

사용 사례: Windows 서버, 완전한 격리가 필요한 서비스

LXC (Linux Containers)

  • 가벼운 OS-레벨 가상화 기술
  • 빠른 속도와 적은 자원 사용
  • 호스트 시스템과 커널 공유
  • 리눅스 기반 애플리케이션만 지원

사용 사례: 웹 서버, 데이터베이스, 개발 환경

Proxmox VE의 주요 특징

웹 브라우저로 쉽게 관리
안정성과 보안성 보장
다양한 저장 방식 지원
백업과 복구 기능 통합
REST API 지원 (자동화 가능, Terrafrom 지원)
실시간 VM/컨테이너 이전 기능
