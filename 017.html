간략한 설치 튜토리얼

설치 순서

1

설치 파일 준비

공식 웹사이트에서 Proxmox VE ISO 파일 다운로드 후
Rufus 또는 Balena Etcher 등으로 부팅 USB 제작

2

설치 프로그램 부팅

제작한 USB로 부팅 후 "Install Proxmox VE" 선택

3

라이선스 동의 (EULA)

최종 사용자 라이선스 계약(EULA) 내용 확인 후 "I agree" 클릭

4

대상 하드디스크 선택

Proxmox VE를 설치할 디스크 선택 (기존 데이터 삭제됨)
Options에서 파일 시스템 (ext4 또는 고급 사용자는 ZFS) 등 설정 가능

5

지역, 시간 및 키보드 설정

국가, 시간대, 키보드 레이아웃 선택 (보통 자동 감지)

6

관리자 암호 및 이메일 설정

root 계정의 암호와 관리자 이메일 주소 입력

7

네트워크 구성

관리 인터페이스, 호스트 이름(FQDN), IP 주소, 넷마스크, 게이트웨이, DNS 서버 정보 입력

8

설치 진행 및 완료

설정 요약 확인 후 "Install" 클릭 (수 분 소요)
설치 완료 후 자동 재부팅 (부팅 USB 제거)

설치 후 주요 작업

1
웹 인터페이스 접속

웹 브라우저에서 https://[Proxmox_IP]:8006 입력
사용자명: root, 암호: 설치 시 설정한 암호
(주의: HTTPS 연결, 자체 서명 인증서 경고 수락)

2
업데이트 저장소 설정

Datacenter > 노드 선택 > Updates > Repositories 이동
유료 구독용 'enterprise' 저장소는 Disable,
무료용 'No-Subscription' 저장소는 Enable
Datacenter > 노드 선택 > Updates > Refresh 후 Upgrade 실행

3
첫 VM 생성 준비 (ISO 업로드)

Datacenter > 노드 선택 > ISO Images (local 스토리지) 선택
'Upload' 버튼으로 OS 설치용 ISO 이미지 파일 업로드
(예: Ubuntu, Windows Server ISO 파일)

* 이후 우측 상단 'Create VM' 버튼으로 가상머신 생성 진행

Proxmox VE 설치 팁

  • 설치 전 서버 하드웨어 호환성을 Proxmox 커뮤니티에서 확인하세요.
  • 네트워크 설정 시 고정 IP 사용을 권장합니다.
  • 설치 후 시스템 시간 동기화(NTP) 설정을 확인하세요.
  • 문제 발생 시 공식 커뮤니티 포럼이 유용합니다.
